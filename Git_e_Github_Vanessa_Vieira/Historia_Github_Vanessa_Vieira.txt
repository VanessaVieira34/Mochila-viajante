Quem inventou o sistema de controle de versão Git e por quê?
R: O Git foi criado po Linus Torvals, com o proposito de unificar projetos, códigos e softwares, onde outros desenvolvedores pudessem trabalhar juntos de maneira inteligente e eficiente.
Quem atualmente é o dono do Github e por quê?
R: A Microsoft, por ser um sistema que unifica muitos recursos e de facil acesso.
Existe outra maneira, além do terminal, de trabalhar com o Github?
R: Sim, pela interface gráfica.
